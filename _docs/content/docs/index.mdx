# Welcome to Surf

Surf is a lightweight, high-performance HTTP web framework for Go. Radix-tree
routing, two handler models (stdlib-compatible and a pooled fast path), a small
standard middleware suite, and zero third-party dependencies.

## Features

<Cards>
  <Card title="Fast Routing" icon="zap">
    Radix-tree router with O(log n) matching — competitive with chi and gin/echo.
  </Card>
  <Card title="Two Handler Models" icon="layers">
    Standard `func(w, r) error` for the stdlib ecosystem; opt-in `func(c *Context) error` for the hottest endpoints.
  </Card>
  <Card title="Errors as Responses" icon="alert-triangle">
    Return an `*HTTPError` and surf renders a JSON envelope; `Abort` stops silently; pre-written responses are preserved.
  </Card>
  <Card title="Typed Services" icon="package">
    `Provide[T]` / `Service[T]` — type-keyed DI with no silent zero-value bugs.
  </Card>
  <Card title="Body Binding" icon="file-input">
    `Bind` / `BindAndValidate` with size limits and a `Validator` hook.
  </Card>
  <Card title="WebSockets" icon="radio">
    RFC 6455 `Upgrade` with same-origin policy by default.
  </Card>
  <Card title="SPA Serving" icon="folder">
    `App.SPA` over any `fs.FS` (incl. `embed.FS`), with immutable asset caching.
  </Card>
  <Card title="Metrics" icon="bar-chart">
    Dependency-free Prometheus exposition middleware.
  </Card>
</Cards>

## Quick Example

```go title="main.go"
package main

import (
    "net/http"

    "github.com/getangry/surf"
)

func main() {
    app := surf.NewApp()

    app.Use(surf.RecoveryWithDefaults())
    app.Use(surf.CORSWithDefaults())

    // Standard handler — stdlib-compatible
    app.Get("/hello/:name", func(w http.ResponseWriter, r *http.Request) error {
        return surf.JSONData(w, map[string]string{
            "message": "Hello, " + surf.Param(r, "name") + "!",
        })
    })

    // Fast-path handler — pooled *Context, ~zero framework allocations
    app.Handle("GET", "/healthz", func(c *surf.Context) error {
        return c.String(http.StatusOK, "ok")
    })

    app.Serve()
}
```

## Why Surf?

- **Simple** — clean API that reads like idiomatic `net/http`.
- **Fast** — competitive with chi (standard path) and approaching gin/echo
  (fast path). See [Performance](/docs/performance).
- **Safe by default** — generic 500 for unhandled errors (no detail leak),
  same-origin WebSocket upgrades, body-size limits on JSON binding.
- **Additive** — features evolve without breaking existing callers.
