# API Reference

Complete reference for all Surf functions and types.

## Application

### NewApp

Creates a new Surf application.

```go
func NewApp(options ...Option) *App
```

**Options:**

```go
// Custom logger
app := surf.NewApp(surf.WithLogger(slog.Default()))

// Custom server configuration
app := surf.NewApp(surf.WithServerConfig(surf.ServerConfig{
    Addr:           ":3000",
    ReadTimeout:    15 * time.Second,
    WriteTimeout:   15 * time.Second,
    IdleTimeout:    60 * time.Second,
    MaxHeaderBytes: 1 << 20,
}))

// Individual options
app := surf.NewApp(
    surf.WithAddr(":3000"),
    surf.WithReadTimeout(30 * time.Second),
    surf.WithWriteTimeout(30 * time.Second),
    surf.WithIdleTimeout(120 * time.Second),
    surf.WithMaxHeaderBytes(2 << 20),
    surf.WithContext(ctx),
    surf.WithShutdownChannel(shutdownChan),
)
```

### Available Options

| Option | Description |
|--------|-------------|
| `WithLogger(logger)` | Custom slog.Logger |
| `WithServerConfig(config)` | Complete ServerConfig |
| `WithAddr(addr)` | Server listen address |
| `WithReadTimeout(d)` | Request read timeout |
| `WithWriteTimeout(d)` | Response write timeout |
| `WithIdleTimeout(d)` | Keep-alive idle timeout |
| `WithMaxHeaderBytes(n)` | Max header size |
| `WithContext(ctx)` | Custom base context |
| `WithShutdownChannel(ch)` | Custom shutdown signal channel |
| `WithCancelFunc(cancel)` | Custom context cancel function |

### Option Type

```go

```

## Request Context Storage

```go
// Basic storage (uses *http.Request directly)
func Store(r *http.Request, key string, value interface{})
func Get(r *http.Request, key string) (interface{}, bool)
func GetString(r *http.Request, key string, defaultVal string) string
func GetInt(r *http.Request, key string, defaultVal int) int
func Delete(r *http.Request)
func MustGet(r *http.Request, key string) interface{}

// Pointer-based storage (for modifying request)
func Set(r **http.Request, key string, value interface{})
func SetMultiple(r **http.Request, values map[string]interface{})
func SetRequestID(r **http.Request, id string)
func SetUserID(r **http.Request, userID string)

// Getters
func GetRequestID(r *http.Request) string
func GetUserID(r *http.Request) string

// Fluent interface
func With(r **http.Request) *WithRequest
func (wr *WithRequest) Set(key string, value interface{}) *WithRequest
func (wr *WithRequest) SetRequestID(id string) *WithRequest
func (wr *WithRequest) SetUserID(userID string) *WithRequest
```

### Usage Example

```go
// Using fluent interface
app.Get("/users/:id", func(w http.ResponseWriter, r *http.Request) error {
    surf.With(&r).
        SetRequestID("req-123").
        SetUserID("user-456").
        Set("operation", "get_user")

    // Later retrieve values
    userID := surf.GetUserID(r)
    requestID := surf.GetRequestID(r)
    return nil
})
```
